# 奇妙集市部署文档

## 1. 环境要求
- Linux 服务器：Ubuntu 22.04 / Debian 12 / CentOS Stream 9 均可。
- PHP：8.2 及以上。
- MySQL：5.7+ 或 8.0+。
- Redis：6.0+。
- Nginx：1.20+。

## 2. PHP 扩展
至少启用以下扩展：
- `pdo_mysql`
- `openssl`
- `curl`
- `mbstring`
- `json`

## 3. 目录结构
- Web 根目录建议指向：`/path/to/mall/public`
- 上传目录：`public/assets/images/uploads`
- 日志目录：`storage/logs`
- Session 目录：`storage/sessions`

## 4. 初始化步骤
1. 创建代码目录并上传项目文件。
2. 复制 `.env.example` 为 `.env`，填写商城库、会员库、Redis、域名配置。
3. 执行数据库脚本：
   ```bash
   mysql -uroot -p < database/init_all.sql
   ```
4. 确保以下目录有写权限：
   ```bash
   chmod -R 775 storage public/assets/images/uploads
   ```

## 5. Nginx 配置示例
```nginx
server {
    listen 80;
    server_name mall.example.com;
    root /var/www/mall/public;
    index index.php;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        include fastcgi_params;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        fastcgi_pass unix:/run/php/php8.2-fpm.sock;
    }

    location ~* \.(jpg|jpeg|png|gif|webp|svg|css|js|json)$ {
        expires 7d;
        access_log off;
    }
}
```

```env

```

## 7. 默认账号
初始化脚本已写入两个账号：
- 管理员：`admin / Admin@123`
- 普通用户：`demo_user / User@123`

## 8. 微信配置说明
### 8.1 微信服务号
在后台“系统设置”页签中填写：
- `app_id`
- `app_secret`
- 模板消息相关配置

### 8.2 微信支付
在后台“系统设置”页签中填写：
- `app_id`
- `merchant_id`
- `merchant_serial_no`
- `api_v3_key`
- `private_key_content`
- `notify_url`

## 9. 定时任务
订单超时自动关闭建议通过 Cron 定时执行：
```bash
* * * * * /usr/bin/php /var/www/mall/scripts/order_timeout_worker.php >/dev/null 2>&1
```

## 10. 图片补充
代码中大量图片路径已预留为 `.webp`，并在对应目录提供 `.webp.txt` 提示词文件。后续只需按提示词生成同名图片并覆盖即可。

## 11. 自检建议
上线前依次检查：
1. 会员库配置测试是否通过。
2. 微信支付配置测试是否通过。
3. 微信服务号配置测试是否通过。
4. 余额支付是否能同步写入会员明细与日志。
5. Redis 是否可用，超时订单是否能自动关闭并回滚库存。
